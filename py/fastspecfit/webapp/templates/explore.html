Fujilupe Data Release


Welcome to the data portal for the DESI FastSpecFit value-added catalogs! FastSpecFit is a stellar continuum and emission-line modeling code for DESI optimized for speed and simplicity, which can be called independently to model the broadband photometry (fastphot) or the three-camera optical spectroscopy from DESI (fastspec).

Please refer to the FastSpecFit documentation for full details regarding this data release, particularly the required acknowledgments and conditions of use.
